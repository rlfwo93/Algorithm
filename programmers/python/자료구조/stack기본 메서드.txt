스택

인스턴스변수

items = [] -> 스택을 담을 배열
top = -1 -> 스택 포인터

메소드(총 10개)
- push - 데이터넣는 메소드
- pop - 데이터를 삭제하고 값을 반환시켜주는 메소드
- indexOf - 원하는 데이터의 위치를 찾아줌(위치를 못찾았을시 -1반환)
- size - 현재 스택에 저장된 요소(데이터)의 총 갯수
- clear - 스택을 초기화 하는 메소드
- is_empty - 스택이 비어있는지 확인(스택이 비어있으면-true 스택에 값이있으면 =false)
- is_full - 스택이 가득 차 있는지 확인(스택이 가득 차있으면 true  가득 안 차있으면 false)
- capacity - 스택의 용량(최대용량) 확인
- dump - 스택 안의 모든 데이터 바닥 -> 꼭대기 순으로 출력
- peek - 맨 꼭대기의 스택 데이터 확인
